# BuildX-Analytics-AdityaDevB

BuildX Analytics Project
Dataset Selected

College Placement Records (Campus Placement Dataset)

Why I Selected This Dataset

I selected this dataset because it helps analyze the factors affecting campus placements, such as academic performance, work experience, specialization, and salary. It provides meaningful insights into student placement trends.

Three Business Questions Answered
How many students were placed and not placed?
Which specialization has the highest average salary?
What is the average salary offered to placed students?


Tools Used

Python (Pandas)

SQLiteOnline

Power BI Desktop

GitHub


Key Insight

Students specializing in Marketing & Finance generally receive higher average salary packages compared to other specializations.
